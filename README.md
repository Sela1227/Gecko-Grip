## 簡介

Leitner 五盒間隔重複演算法，幫助你高效記憶英文單字。
看英文 → 想中文 → `Space` 揭示 → 評分，答對的拉長複習間隔，答錯的明天再考。

雙詞庫：9,858 個通用英文高頻字（Google 兆詞語料庫詞頻排序）+ 252 個醫學專業詞彙（病房臨床 / 檢驗影像 / 腫瘤科 / 放射腫瘤 / 研究論文），兩庫各自獨立追蹤進度。

發音優先播放字典真人錄音，離線或查無音檔時改用系統語音。

按 `T` 可加做拼字測試 — **開啟時上方英文會反向遮蔽成底線**（`_ _ _ _ _`，只留字數提示），憑記憶輸入、確認後才揭曉正確拼字並比對。

## 使用方式

雙擊 `index.html`，或部署到 GitHub Pages。無需安裝、無帳號、完全離線可用。

```bash
# 本機預覽（推薦）
python -m http.server 8000
# 開 http://localhost:8000
```

## 操作快捷鍵

| 鍵 | 動作 |
|---|---|
| `Space` | 顯示中文意思 |
| `1` | 不認得（重回 Box 1，之後每日出現直到答對） |
| `2` | 認識（升一盒，間隔拉長） |
| `P` | 播放發音 |
| `T` | 展開／收起拼字測試（開啟時英文遮蔽） |

## Leitner 五盒

| 盒 | 複習間隔 |
|---|---|
| Box 1 | 1 天 |
| Box 2 | 3 天 |
| Box 3 | 7 天 |
| Box 4 | 14 天 |
| Box 5 | 30 天（精通） |

### 答錯的字會被記住

答錯不只是打回 Box 1 而已：

- **當日補考**：隔 4 張卡後在同一回合再考一次
- **補考答對不給升級**：你 4 張卡前才看過答案，那是短期記憶 — 仍停在 Box 1，明天重考
- **間隔永久縮短**：曾錯過的字，之後每一盒的間隔都會依累計錯誤次數打折

| 累計答錯 | 間隔係數 | Box 5 實際間隔 |
|---|---|---|
| 0 次 | ×1.0 | 30 天 |
| 1 次 | ×0.7 | 21 天 |
| 2 次 | ×0.55 | 17 天 |
| 3 次以上 | ×0.45 | 14 天 |

也就是說 **錯過的字永遠不會真的「畢業」** — 就算爬到 Box 5，它仍會每兩三週回來找你一次。
