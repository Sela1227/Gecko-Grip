## 簡介

Leitner 五盒間隔重複演算法，幫助你高效記憶英文單字。
看英文 → 想中文 → `Space` 揭示 → 評分，答對的拉長複習間隔，答錯的明天再考。

雙詞庫：9,858 個通用英文高頻字（Google 兆詞語料庫詞頻排序）+ 252 個醫學專業詞彙（病房臨床 / 檢驗影像 / 腫瘤科 / 放射腫瘤 / 研究論文），兩庫各自獨立追蹤進度。

### 新字依詞性配比選取

動詞決定一句話的核心意思，但詞頻表 rank 101–3000 實測有 **59% 是名詞、只有 19% 是動詞**（Brown 語料庫統計）—— 照詞頻順序背，六成時間在背名詞。

| 方案 | 動詞 | 名詞 | 形容詞 | 副詞 |
|---|---|---|---|---|
| 醫學論文／國際會議（預設） | 45% | 35% | 20% | — |
| 均衡 | 30% | 40% | 20% | 10% |
| 依詞頻 | 不分詞性，照 rank 順序 | | | |
| 自訂 | 自行設定 | | | |

同一桶內仍照詞頻由高到低。首頁會顯示「已學組成」讓你驗證配比真的生效。

詞性判定用 Brown 語料庫（116 萬個已標註詞）的實際使用頻率，不是字典的詞性排列順序。變化形（`said` / `years` / `is` / `using`）佔詞頻表 25.2%，一律不發新卡 —— 學 `say` 就好，不用另外背 `said`。

發音優先播放字典真人錄音，離線或查無音檔時改用系統語音。

按 `T` 可加做拼字測試 — **開啟時上方英文會反向遮蔽成底線**（`_ _ _ _ _`，只留字數提示），憑記憶輸入、確認後才揭曉正確拼字並比對。

## 使用方式

雙擊 `index.html`，或部署到 GitHub Pages。無帳號、無後端。

### 安裝為 App（PWA）

經 https 或本機伺服器開啟後，可從瀏覽器選單「加入主畫面」安裝，之後像原生 App 一樣從桌面啟動、全螢幕執行。

**離線可完整練習** —— 詞庫、釋義、排程全部內嵌。需要連線的只有字典真人錄音與英文定義；離線時自動改用系統語音，中文釋義與 Leitner 排程完全不受影響。

```bash
# 本機預覽（推薦）
python -m http.server 8000
# 開 http://localhost:8000
```

## 操作快捷鍵

| 鍵 | 動作 |
|---|---|
| `Space` | 顯示中文意思 |
| `1` | 不認得（重回 Box 1，之後每日出現直到答對） |
| `2` | 認識（升一盒，間隔拉長） |
| `P` | 播放發音 |
| `T` | 展開／收起拼字測試（開啟時英文遮蔽） |

## Leitner 五盒

| 盒 | 複習間隔 |
|---|---|
| Box 1 | 1 天 |
| Box 2 | 3 天 |
| Box 3 | 7 天 |
| Box 4 | 14 天 |
| Box 5 | 30 天（精通） |

### 答錯的字會被記住

答錯不只是打回 Box 1 而已：

- **當日補考**：隔 4 張卡後在同一回合再考一次
- **補考答對不給升級**：你 4 張卡前才看過答案，那是短期記憶 — 仍停在 Box 1，明天重考
- **間隔永久縮短**：曾錯過的字，之後每一盒的間隔都會依累計錯誤次數打折

| 累計答錯 | 間隔係數 | Box 5 實際間隔 |
|---|---|---|
| 0 次 | ×1.0 | 30 天 |
| 1 次 | ×0.7 | 21 天 |
| 2 次 | ×0.55 | 17 天 |
| 3 次以上 | ×0.45 | 14 天 |

也就是說 **錯過的字永遠不會真的「畢業」** — 就算爬到 Box 5，它仍會每兩三週回來找你一次。
